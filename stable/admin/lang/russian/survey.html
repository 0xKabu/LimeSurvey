Администрирование

ОПРОС:
Вы находитесь в разделе администрирования опроса PHPSurveyor.

Отсюда Вы можете просматривать и управлять основными настройками Вашего опроса. Так же Вы можете добавлять новые группы или выбирать существующие для просмотра и изменения .

Кнопки:
Опрос активен: Эта иконка отображается, если опрос автивирован и готов для сохранения ответов.
Де-активировать Опрос: Нажмите на эту кнопку для остановки активного опроса
Опрос не активен: Эта иконка отображается если опрос не активен
Активировать Опрос: Нажмите сюда для активации неактивного опроса
Провести/Тестировать Опрос: Для запуска опроса нажмите на эту кнопку (будет проведено тестирования если опрос не активен)
Ввод данных: Кликните сюда для прехода к экрану ввода данных. Кнопка предназачена для воода данных с заполненных печатных анкет
Печатная версия: Кликните сюда для просмотра "печатной" версии опроса.
Редактирование опроса: Кликните сюда для изменение деталей текущего опроса.
Удаление опроса: Кликните сюда для удаления опроса. Кнопка отображается только если в опросе нет групп.
Экспорт опроса: Кликните сюда для выгрузки данных опроса, ключая все группы, вопросы, отвты и условия, в файл.
Посмотрель результаты: Кликните сюда для просмотра ответов опроса. Кнопка доступна только для активных опросов.
Ключевые фразы: Кликните сюда для упавления ключевыми фразами текущего опроса.
Добавить группу: Кликните сюда для добавления новой группы вопросов.

Добавление новой группы
Нажмите на для добавления новой группы вопросов в опрос.

Любой опрос должен иметь хотя бы одну группу. Группы используются для группировки логически связанных вопросов. Группа имеет имя и описание. После выбора "Добавить группу" Вам выдается экран с двумя полями для заполнения. Имя группы: Заголовок группы. Это поле должно быть заполнено. Оноо так же определяет порядок, в котором выдаются группы при выполнении опроса. Группы упорядочены по алфавиту по Имени группы. Поэтому старайтесь называть группы в алфавитном порядке. Например: "Раздел 1:) О Вас." , "Раздел 2:) Ваше видение.".  Описание группы: Это поле не обязательно для заполнения. Оно может быть использовано для отображения описания группы пользователю при проведении опроса. Если описания нет, то участнику вопросы отображаются в формате "все за раз", без показа отдельных страниц с именами групп и описаниями.
