Вы находитесь в разделе управления пользователями PHPSurveyor.

Здесь Вы можете посмотреть и изменить настройки безопастности и права доступа пользователей.

Настройки безопастности PHPSurveyor связаны с системой безопасности каталогов Apache. Если Вы используете в качестве Web-сервера не Apache, то Вы не сможете использовать эту систему безопастности, и Вы должны инициализировать безопасность каталогов с использованием методов Вашего сервера.

Инициализация Безопастности
При первом запуске системы безопастности из PHPSurveyor будут инициализированы установки безопастности. При инициализации PhpSurveyor найдет программу "htpasswd" из Apache и создаст пользовательский файл. Так же будет создан файл 'htaccess' в каталоге admin для ограничения доступа к административным функциям для пользователей, упомянутых в файле htpasswd. Начальные имя пользователя и пароль определяются настройками 'default' (по умолчанию) в файле config.php. После успешной инициализации системы безопасности Ваш броузер будет спрашивать имя пользователя и пароль для доступа к административной части PHPSurveyor.

Добавление пользователей
После инициализации системы безопасности и в входа в административную часть Вы можете просмотрет существующих пользователей системы, добавить новых, удалять пользователей или изменять их данные. Это делается очень просто. В отображенном списке пользователей нажмите кнопку "изменить" для исправления данных или "Удалить" для удаления пользователя. Если в списке представлен только один пользователь, то Вы не сможете его удалить.

Выключение системы безопасности
Для выключения системы безопастности PHPSurveyor нажмите на кнопку "Выключить систему безопастности". При этом удалятся файлы htpasswd, htaccess и информация о пользователях каталога.

Ограничения системы безопасности PHPSurveyor
До реализации более гибкой системы безопасности необходимо помнить, что настройки безопасности PHPSurveyor дествуют по принципу 'все-или-ничего'. Пользователи либо имеют доступ ко ВСЕМ административным функциям либо ни к каким. Система спланирована таким образом, чтобы со временем позволит Вам настраивать уровни доступа. Заметьте что все возможности системы безопасности, включая пароли в виде плоского текста, доступны всем пользователям.

Разрешение проблем
Если у Вас проблемы с установкой безопасности:

1.

Проверьте наличие прав на чтение/запись в каталоге admin. PHPSurveyor требует возможности создания фалов .htpasswd и .htaccess в этом каталоге.

2.

Проверьте доступность программы htpasswd. В частности для windows сервера, проще всего поместить копию программы в каталог admin. На Linux серверах Phpsurveyorservers будет пытаться искать htpasswd с использованием "find".
