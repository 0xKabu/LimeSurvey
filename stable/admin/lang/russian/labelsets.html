Администрирование набора меток:
Вы находитесь в секции Администрирование набора меток PHPSurveyor.

Отсюда Вы можете создавать, изменять и удалять наботы меток, используемы в вопросах "с гибкими метками". Наборы меток позволяют Вам определить заранее серии предопределенных для выбора ответов. э Например, в опросе Вы можете порость ответить участников о частном суждении или сценарии, дав им выбрать ответ для каждого утверждения из готового ответов. Пример приведен ниже:

Пожалуйста, отметьте высказывания, которые больше всего соответсвует следующим утверждениям:
  Полностью согласен Частично согласен Ничего не могу сказать Частично не согласен Полностью не согласен
Я очень странная личность
Я люблю мороженное
Зеленый - мой любимый цвет

Используя наборы меток Вы можете создавть собственные заголовки колонок для подобных вопросов.

Следующие возможности доступны на панели Администрирования Наборов меток:

Начало Кликните сюда для возврата к главному административному экрану PHPSurveyor
Добавить набор меток: Кликните сюда для создания нового набора меток или для импорта набора меток из файла


Добавление нового набора меток
Кликните на для создания или импорта набора меток.

Для набора меток обязательно только имя. Каждый набор меток автоматически получает уникальный идентификатор "LID" (индекс набора меток) . Вы можете создать набор меток с тем же именем, о обычно это не рекомендуется во избежания ошибок . Набор меток отличается от друго с таким же именем по его идентификатору LID.

Импорт набора меток
Вы можете использовать возможность импорта набора меток (внизу экрана). Кликните на просмотр, выберите *.sql файл PHPSurveyor, содержащий набор меток и нажмите кнопку Импорт (Import).
