Вы находитесь в секции администрирования ответов PHPSurveyor.

Находясь в это секции Вы можете просматривать, редактировать и изменять порядок ответов для конкретного вопроса Вашего опроса.

Кнопки:
Перемещает ответ вверх: Нажмите на эту кнопку для перемещения этого ответа вверх при отображении.
Перемещает ответ вниз: Нажмите на эту кнопку для перемещения этого ответа вниз при отображении.
Сохранение изменений в ответе: Нажмите на эту кнопку для сохранения любых изменений, сделанных в ответе.
Удаление ответа: Нажмите на эту кнопку для того чтобы удалить этот ответ.
Добавление ответа: Нажмите на эту кнопку для добавления нового ответа.
Исправляет порядок сортировки: Нажмите на эту кнопку для исправления каких бы то нибыло проблем с сортировкой ответов.
