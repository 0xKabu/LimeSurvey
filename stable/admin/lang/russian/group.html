Вы находитесь в секции администрирования групп PHPSurveyor.

Отсюда Вы можете просматривать и управлять группами Вашего опроса. Вы так же может здесь добавить новый вопрос к выбранной группе или выбрать существующий вопрос из группы для просмотра или редактирования.

Кнопки:
Редактирование группу: Нажмите эту кнопку для редактирования деталей текущей группы.
Удаление группы: Нажмите эту кнопку для удаления текущей группы. Эта кнопка отображается только если в группе нет вопросов.
Добавить вопрос: Нажмите эту кнопку для добавления нового вопроса в группу. Кнопка будет недоступна, если опрос активен.

Добавление нового вопроса
Нажмите на чтобы добавить новый вопрос в Ваш опрос.

Вы можете добавить произвольное количество вопросов в каждую группу опроса. При нажатии на кнопку "Добавить новый вопрос", Вам будет предъявлен экран, позволяющий ввести:
Код вопроса: обязательное поле, используемое для опредления порядка следования вопросов в группе. Код вопрос может быть до 5 символов в длину. Вопросы упорядычиваются по коду в алфавитнгом порядке.
Вопрос: поле произвольной длины. Содержит вопрос предлагаемы участнику опроса.
Помощь: в этом поле Вы можете указать поясняющий текст к вопросу и ответам. Он будет отбражать ниже возможных ответов.э
Тип вопроса: поле с выпадающим списком. Содержит различные типы доступных предопределенных вопросов. Детальное их описание Вы найдете в руководстве пользователя.
Обязательность: позволяет предотвратить продвижение по опросу без ответа на указанный вопрос.
Другое: для некоторых типов вопроса позволяет добавить помимо ответов типа "Да"/"Нет" добавить дополнительный ответ "не знаю"/"без ответа".
